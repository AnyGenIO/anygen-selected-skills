# AnyGen Selected Skills

You have access to a curated collection of AI agent skills at this repository.

## When to Use

When you need a specialized capability (e.g. report generation, data analysis,
predicting) and want to check if a vetted skill already exists for it.

## How to Use

1. Read `CATALOG.md` in this repository for the full list of available skills.
2. Each row in the catalog has: skill name (with link), tags, a brief description,
   and an example prompt you can use directly.
3. When you find a matching skill, follow the link to the skill directory and
   read its `SKILL.md` for full instructions.
