### 001 · Forecast Intelligence

| | |
|---|---|
| **Skill** | `kasparchen:geopolitical-forecast` |
| **Author** | [@KasparChen](https://github.com/KasparChen) |
| **Domain** | Geopolitics, Finance, Technology, Corporate |
| **Showcase** | [ClawHub](https://clawdhub.com/skill/kasparchen/geopolitical-forecast) |
| **Directory** | [`selected-skills/kasparchen:geopolitical-forecast/`](selected-skills/kasparchen:geopolitical-forecast/) |

**What it does:**
Generates professional CIA-brief-style forecast intelligence reports as PDF. The agent researches a topic via web search, formulates probability-ranked predictions, builds D3 visualizations, cross-references prediction markets, and outputs a polished single-page intelligence assessment.

**Why we picked it:**
- Solves a real problem — turns hours of analyst work into a 2-minute automated workflow
- Beautiful output — white monochrome design with 10 built-in D3 visualization types
- Rigorous methodology — enforces source verification, causal logic in drivers, and probability calibration
- Truly agent-agnostic — works with any agent that can search, read/write files, and run shell commands
- Zero bloat — only needs Playwright for PDF conversion; everything else is self-contained HTML

**Sample workflow:**
```
User: "Will there be a US-Iran ceasefire before April 2026?"
Agent: Reads SKILL.md → Researches 8+ sources → Builds predictions →
       Generates HTML with map + entity graph → Converts to PDF
Output: Professional 1-page intelligence brief with 34% probability verdict
```
